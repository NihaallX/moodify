After Adding Screenshots

After placing your screenshots in the correct folders, run these commands:

  1. npm run build - to build the app with your new screenshots
  2. npm run deploy - to deploy the updated landing page to GitHub Pages

Landing Page Update

The landing page HTML is already configured to use these files. You just need to place your screenshots in the correct locations with these exact filenames.
